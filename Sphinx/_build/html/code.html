class UnitTests.TestFileIO(methodName='runTest')¶

A class for unit testing the input/output of PyDPD.

testFileLines()¶

This test compares the output of two results which should have the same value. These values are the results of the following operations: - Reading the file into a string. - Calling PyDPD.getFileLines() to get a List of all lines in the file. Then

adding all lines in the List together.

The test file used is a Javascript file from the project JS-Dataset-Parser which was downloaded from “https://github.com/mikesiers/JS-Dataset-Parser”.

Parameters:self – A parameter required for Python unit testing.
