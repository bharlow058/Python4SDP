class UnitTests.TestFileIO(methodName='runTest')¶

A class for unit testing the input/output of PyDPD.

testFileLines()¶

This test checks the equality of two strings. The first string is generated by storing the file as a string. The second string is generated by calling PyDPD.getFileLines() to get a List of all lines in the file, then adding all lines in this List together. The test file used is a Javascript file from the project JS-Dataset-Parser which was downloaded from “https://github.com/mikesiers/JS-Dataset-Parser”.

Parameters:self – A parameter required for Python unit testing.
